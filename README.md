# Конвертер азбуки Морзе

Pet-проект на Go, реализующий веб-сервис для конвертации текста в азбуку Морзе и обратно.  
Пользователь загружает файл через HTML-форму, сервис автоматически определяет, содержит ли файл обычный текст или код Морзе, выполняет преобразование и сохраняет результат в новый файл.  
Проект демонстрирует создание HTTP-сервера, обработку multipart/form-data, работу с файлами, а также использование стандартной библиотеки Go.

## Чему вы научитесь, изучив этот проект

- Создавать HTTP-сервер на чистой стандартной библиотеке.
- Парсить HTML-формы и извлекать загруженные файлы.
- Регистрировать обработчики (хендлеры) для разных маршрутов.
- Автоматически определять тип содержимого файла (текст или код Морзе).
- Использовать готовый пакет для конвертации (`morse`).
- Обрабатывать ошибки и устанавливать таймауты для сервера.

## Требования

- Go 1.18 или выше.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/morse-converter.git
   cd morse-converter
   ```

2. Запустите сервер:
   ```bash
   go run cmd/main.go
   ```

3. Откройте браузер и перейдите по адресу `http://localhost:8080`. Вы увидите форму загрузки файла.

## Использование

1. На главной странице (`/`) находится форма для выбора файла.
2. Выберите файл (можно использовать предоставленные `test` или `test12`) и нажмите «Отправить».
3. Сервер определит, является ли содержимое файла текстом или кодом Морзе:
   - Если это текст, он будет преобразован в код Морзе.
   - Если это код Морзе, он будет преобразован в текст.
4. Результат конвертации будет сохранён в новый файл с именем, содержащим временную метку, а также отобразится на странице.

### Примеры тестовых файлов

- `test` – содержит код Морзе (например, `.... . .-.. .-.. ---`).
- `test12` – содержит обычный текст (например, «Привет, мир!»).

После обработки файл с результатом будет доступен для скачивания или просмотра.

## Описание пакетов

### `pkg/morse`
Готовый пакет, предоставляющий две функции:
- `ToText(morse string) string` – конвертирует строку с кодом Морзе в текст на русском языке.
- `ToMorse(text string) string` – конвертирует текст в код Морзе.

### `service`
Содержит функцию `Convert(data []byte) (string, error)`, которая:
- Анализирует входные данные (строку), определяет, что это – обычный текст или код Морзе.
- Вызывает соответствующую функцию из пакета `morse`.
- Возвращает результат конвертации или ошибку, если определение не удалось.

### `handlers`
Два HTTP-обработчика:
- `IndexHandler` – отдаёт HTML-форму (`index.html`) по маршруту `/`.
- `UploadHandler` – обрабатывает POST-запрос на `/upload`:
  1. Парсит multipart-форму.
  2. Извлекает загруженный файл.
  3. Читает его содержимое.
  4. Передаёт данные в `service.Convert`.
  5. Создаёт локальный файл с уникальным именем (на основе `time.Now()` и исходного расширения).
  6. Записывает результат конвертации в новый файл.
  7. Возвращает пользователю строку с результатом.

### `server`
Определяет структуру `Server`, которая объединяет логгер и HTTP-сервер.  
Функция `NewServer(logger *log.Logger) *Server`:
- Создаёт роутер (`http.ServeMux`).
- Регистрирует обработчики из пакета `handlers`.
- Настраивает `http.Server` с портом `8080`, таймаутами и переданным логгером.
- Возвращает указатель на `Server`.

### `cmd/main.go`
Точка входа:
- Создаёт логгер.
- Инициализирует сервер через `server.NewServer`.
- Запускает сервер. В случае ошибки логирует её с уровнем Fatal.

## Пример работы

1. Запустите сервер.
2. Откройте `http://localhost:8080`.
3. Выберите файл `test12` (с текстом) и отправьте.
4. Сервер определит, что это текст, и преобразует его в код Морзе.
5. В ответе вы увидите преобразованную строку, а на диске появится новый файл с именем вроде `2025-03-11 12-30-45.123.txt` (если исходный файл имел расширение `.txt`) или без расширения, содержащий код Морзе.
